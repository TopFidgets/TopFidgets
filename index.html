📝 Comment modifier la boutique
Vos produits, catégories et réglages sont enregistrés directement dans le fichier du site — c'est permanent et visible par tous vos clients. 🎉

Pour les modifier :
  1. Ouvrez votre fichier index.html sur GitHub.
  2. Cliquez sur le crayon ✏️ pour modifier.
  3. Tout en haut du code, trouvez le bloc « VOS PRODUITS ».
  4. Copiez une ligne de produit, collez-la, et changez les valeurs (nom, prix, stock, image...).
  5. Cliquez sur Commit changes pour enregistrer.
Le guide complet (tous les champs expliqués) est écrit en commentaire juste au-dessus de la liste des produits dans le fichier.

💡 Astuce : le mot de passe admin actuel est défini dans le bloc SETTINGS du fichier. Changez-le là-bas si besoin.
